
# Руководство пользователя

## Назначение системы

Чат-бот предназначен для поиска и размещения заявок на взаимопомощь между пользователями социальной сети ВКонтакте.

## Начало работы

Для начала работы необходимо открыть диалог с ботом и отправить команду запуска.

## Основные возможности

Пользователь может:

* создать заявку на помощь;
* просматривать доступные заявки;
* указывать местоположение;
* получать информацию о ближайших заявках.

## Пример сценария использования

1. Пользователь запускает бота.
2. Создает заявку с описанием проблемы.
3. Указывает местоположение.
4. Другие пользователи могут увидеть заявку и откликнуться.

## Ограничения системы

* требуется подключение к интернету;
* необходим аккаунт ВКонтакте;
* корректная работа зависит от VK API.
