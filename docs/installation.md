# Установка и запуск проекта

## Требования

Для работы проекта необходимы:

* Python 3.13 или выше;
* доступ к интернету;
* токен VK API.

## Установка зависимостей

После клонирования репозитория необходимо установить зависимости:

pip install -r requirements.txt

## Настройка проекта

В файле config.py необходимо указать токен сообщества ВКонтакте.

## Запуск проекта

Для запуска необходимо выполнить команду:

python bot.py

или использовать файл start_bot.bat.

## Проверка работы

После запуска бот начинает принимать сообщения от пользователей через сообщество ВКонтакте.
