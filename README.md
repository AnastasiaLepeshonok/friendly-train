# Friendly Neighbor
# Дружелюбный Сосед
Чат-бот для социальной сети ВКонтакте, предназначенный для координации локальной взаимопомощи между жителями района с использованием интерактивного картографического сервиса.

## Основные функции

- Создание заявок на помощь
- Поиск заявок
- Интерактивная карта
- Категории помощи
- Уведомления пользователей
- Система статусов заявок

## Технологии

- Python
- VK API
- SQLite
- Yandex Maps API

## Архитектура системы

Система состоит из:
- VK-бота
- базы данных
- картографического сервиса
- модуля обработки заявок

## Установка

1. Клонировать репозиторий:

git clone ...

2. Установить зависимости:

pip install -r requirements.txt

3. Настроить config.py

4. Запустить:

python bot.py
